LÓGICA COMPUTACIONAL


A lógica computacional é o estudo das regras e princípios que orientam o raciocínio lógico em sistemas computacionais, é pensar como um computador para entender a sua logica por tras disso. É uma disciplina fundamental para a formação em tecnologia.

Importância

- Permite "ensinar" a máquina a "pensar", ajuda a criar algoritmos, programação e tomada de decisões lógicas, contribuindo assim para o desenvolvimento cognitivo e pensamento critico. Melhora a percepção analítica das situações, o que jjuda a ter uma escrita mais concisa, auxiliando na resolução de problemas no dia a dia.

Aplicações

- É uma das disciplinas básicas dos cursos de computação, sistemas e informação ; - É fundamental para a criação de algoritmos e programação ;

- É importante para a tomada de decisões lógicas em sistemas computacionais ; - Etapas do pensamento computacional ;

- Decomposição dos problemas em pequenas partes ; - Reconhecimento de padrões e categorização ;

- Abstração dos problemas ; - Criação de um algoritmo (sequência de operações) para a resolução de problemas.
